# Intern Daily Progress Tracker

A simple Python project for tracking daily progress of interns.

## Features
- Logs daily progress with fields: Date, Tasks, Challenges, Learnings, Goals, Hours
- Saves data to a CSV file (`data/progress_log.csv`)
- Lightweight and easy to automate with GitHub Actions
